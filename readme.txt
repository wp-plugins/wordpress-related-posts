== Description ==

The wp plugin will add related links for posts & pages with your own anchor text

== Installation ==

1. Upload the 'wordpress-related-posts' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create any post or page, where you can see the option to add the related post.

== Frequently Asked Questions ==

= What's the advantage of this plugin over other related posts plugin? =
The advantage of this plugin is, you can give your own anchor text and alt/title text for the related post/page entry, so that you can get more SEO benefits.

= Can I link to a page from a post or vice versa? =
Yes, you can link from/to a page or post but not to other part of your blog.

= What is the major requirement to work this plugin well? =
You must have enabled Javascript in your browser.
