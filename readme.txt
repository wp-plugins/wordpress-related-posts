== Description ==

The wp plugin will add related links for posts & pages with your own anchor text

== Installation ==

1. Upload the 'related-posts.php' file to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the 'Settings' menu and click on 'Related Posts' to see the plugin options.
